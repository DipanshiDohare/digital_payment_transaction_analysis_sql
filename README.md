# 💳 Digital Payment Transaction Analysis using SQL

## 📌 Project Overview

This project analyzes paypal transaction data using MySQL to identify transaction trends, customer behavior, merchant performance, and business insights. The project demonstrates how SQL can be used to clean, manage, and analyze real-world transactional data to support data-driven business decisions.

---

## 🎯 Business Objective

The objective of this project is to:

- Analyze digital payment transactions.
- Understand customer and merchant behavior.
- Identify monthly transaction trends.
- Compare transaction activity across countries and currencies.
- Generate business insights and recommendations using SQL.

## 🧹 Data Cleaning & Preparation

Before performing the analysis, the dataset was cleaned and prepared using SQL.

The following tasks were completed:

- Converted string-formatted dates into the DATE/DATETIME format.
- Updated transaction date values using SQL UPDATE statements.
- Verified data consistency across related tables.
- Prepared the dataset for joins, aggregations, and business analysis.

## 📊 Business Questions Solved

This project answers several business questions, including:

1. Which countries generated the highest transaction value?
2. Which merchants received the highest payments?
3. Which currency recorded the highest transaction value?
4. How did transaction amounts change month by month?
5. Which customer generated the highest transaction value?
6. What percentage of transactions were Domestic vs International?
7. How many transactions were classified as High Value and Regular?
8. What was the average transaction amount received by each merchant?

---

## 🔍 Key Findings

- Zambia recorded the highest total transaction value.
- Gould LLC received the highest total payment among all merchants.
- EUR recorded the highest converted transaction amount.
- Most transactions were categorized as Regular transactions.
- International transactions accounted for the majority of payment activity.
- Monthly transaction values fluctuated throughout the year.
- The highest spending customer generated over **233,000** in total transaction value.

---

## 💡 Business Insights

- A small number of merchants contributed a large share of total transaction value.
- International payments dominated the transaction volume, indicating strong cross-border activity.
- Regular transactions made up the majority of overall business operations.
- Monthly transaction amounts changed over time, suggesting seasonal transaction patterns.
- High-value transactions were limited but represented significant payment amounts.

---

## 📈 Recommendations

- Strengthen relationships with top-performing merchants.
- Continue expanding international payment services.
- Monitor high-value transactions to improve risk management.
- Analyze monthly trends for better forecasting and business planning.
- Reward high-value customers through loyalty or retention programs.
